OpenHENS Integration Task Files
===============================

The editable PRD task checklists for the OpenHENS integration live in:

``docs/developer/openhens-integration-tasks/``

Start with ``README.md`` in that directory. The individual Markdown files are
intentionally kept as source checklists so implementation agents can tick
``- [ ]`` items directly while they work.

Each task file links back to the relevant sections of
``OPENHENS_MIGRATION_PLAN.md`` and lists the settled decisions that agents
should not reinterpret locally.

The task README also embeds the required OpenPinch-owned workflow. Deviations
from that workflow are not permitted in implementation tasks.
