# Domain Entities

- **Prepared problem**: canonical input, configuration, zone tree, and process
  components without derived results.
- **Component inventory**: read-only mapping of component identifiers to
  returned process-MVR component objects.
- **Design view**: application behavior wrapping one serializable HEN synthesis
  result, with ranked network and lazy grid operations.
- **Workspace case**: one named `PinchProblem` and its canonical stored input.
- **Case batch**: ordered names plus mirrored target/design/observation accessors
  and deterministic per-case outcomes.
- **Plot accessor**: read-only selection and rendering of cached graph records.
