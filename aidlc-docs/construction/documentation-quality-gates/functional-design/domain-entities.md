# Domain Entities

- **Public owner**: a supported problem, workspace, accessor, or result-view
  class whose public members form part of the denominator.
- **Operation row**: one owner and operation with classification, semantic
  mode, primary tutorial, execution profile, optional dependency, and status.
- **Tutorial owner**: one packaged notebook containing an executable reference
  to the operation.
- **Execution profile**: `base`, `slow-hpr`, `solver`, or `interactive`.
- **Coverage result**: exact live-to-manifest parity plus executable-cell
  ownership, reported independently from optional-profile execution evidence.
