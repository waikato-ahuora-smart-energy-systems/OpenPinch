# HEN NFR Requirements

- No solver may silently average segmented CP.
- Integer solvers use explicit interval selection; continuous solvers use a bounded active-set iteration.
- Failed active-set convergence produces a deterministic capability error.
- Parent topology size remains independent of segment count; only thermal detail scales.
- Every extracted result is verified from public network data.
- Constant-CP HEN behavior remains a one-segment specialization.
